# AI Decision Re-checkability Audit Trail

This example demonstrates how to build an AI-powered system with re-checkability and transparency. It simulates a credit decision process where each decision is accompanied by a detailed audit log. This log includes the exact input data, the final decision, human-readable reasons, and structured factors that influenced the outcome, allowing independent verification and understanding of the AI's logic, crucial for reliability.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
